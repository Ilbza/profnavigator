# ПрофНавигатор

## Что уже есть
- Профориентация по баллам и интересам.
- Фильтры по региону и формату (бюджет/платно).
- База направлений из проверенных источников в `data/programs-db.js`.
- Экспорт PDF-заявки.

## Обновление базы
1. Откройте терминал в папке проекта.
2. Запустите:

```bash
node scripts/update-db.mjs
```

Скрипт:
- парсит источники и обновляет список направлений автоматически;
- обновляет дату/версию базы;
- если источник временно не распарсился, сохраняет предыдущие записи (fallback), чтобы не терять покрытие.

## Регулярное обновление (пример cron, ежедневно в 06:30)
```bash
30 6 * * * cd "/Users/ilbza/Documents/New project" && /usr/bin/env node scripts/update-db.mjs
```
